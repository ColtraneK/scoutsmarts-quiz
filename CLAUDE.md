# ScoutSmarts Merit Badge Quiz

## What this is
A 10-question quiz that uses OpenAI gpt-4.1 to recommend personalized BSA merit badges.
Built by Cole, founder of ScoutSmarts (scoutsmarts.com), an Eagle Scout.
Live at: https://merit-badge-quiz.scoutsmarts.com

## Tech stack
- **Frontend**: Vite + React 19 (single file: `src/App.jsx`)
- **Backend**: Netlify serverless function (`netlify/functions/generate.js`)
- **APIs**: OpenAI (gpt-4.1), Kit/ConvertKit (v4 subscriber API), Resend (email)
- **Hosting**: Netlify, auto-deploys on push to `master`
- **Repo**: https://github.com/ColtraneK/scoutsmarts-quiz

## File structure
```
src/App.jsx              — Entire frontend (quiz, email gate, results screen)
netlify/functions/generate.js — Serverless function (OpenAI + Kit + Resend)
index.html               — OG tags, fonts, favicon
public/                   — Static assets (favicon, logo, og-image)
```

## How the quiz works
1. 10 questions: age, interests, environment, challenge level, priorities, etc.
2. Age 10-12: parent consent check, no email required
3. Age 13+: email gate collects email before showing results
4. Email + answers → serverless function → OpenAI returns JSON with:
   - `scout_type`, `personality_description`, 10 `badges`, `eagle_tip`
5. Function also (if email provided):
   - Creates Kit subscriber with custom fields (scout_type, age_range, interests, top_badges, challenge_level, environment)
   - Sends results email via Resend from cole@scoutsmarts.com
6. Results displayed on screen with expandable badge cards

## Netlify env vars (already set)

Each subscriber gets: `scout_type`, `age_range`, `interests`, `top_badges`, `challenge_level`, `environment`

## Badge guide URLs
Each recommended badge links to: `https://scoutsmarts.com/{badge-slug}-merit-badge-guide/`
Slug generated by: `name.toLowerCase().replace(/[^a-z0-9]+/g, "-")`
There are 136 active BSA merit badges in the database.

## Key decisions made
- ConvertKit integration is server-side (not client-side) via Kit v4 API
- Side effects (Kit + Resend) are `await`ed before response returns
- CSS `zoom: 1.33` on desktop (768px+) for larger default layout
- Time estimates start at "1-2 weekends" (not "A weekend" — unrealistic)
- Removed discontinued/fake badges: American Indian Culture, Health Care Professions, Scuba Diving
- "Small-Boat Sailing" uses official BSA hyphenation
